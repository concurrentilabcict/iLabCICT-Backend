# iLabCICT Backend API

Backend RESTful API for the **Laboratory Maintenance and Technical Support Management System** of the College of Information and Communications Technology (CICT) at Bulacan State University.

---

## Overview

iLabCICT is a centralized maintenance and technical support management system designed to streamline laboratory operations within CICT laboratories. The system enables students, faculty, laboratory personnel, and administrators to efficiently submit, track, assign, and resolve technical issues and maintenance requests.

This repository contains the backend API built using:

- Django
- Django REST Framework
- PostgreSQL
- Neon Database

---

# Architecture

- RESTful API Architecture
- Feature-based Layered Architecture
- JSON-based Communication
- JWT Authentication
